HTML -- Hypertext markup language

-- Used to create webpages
-- Widely used in webpages
-- Just a markup language it is a programming language

DOCTYPE -- This tag specifies the laguage we write on the page
<html> -- this tag signals that from here we are going to write HTML code

<head> -- Here we write metadata for the pages mostly meant for search engines

<title> -- Title of the webpage, appear in tab bar head

Header tags -- <h1> to <h6>, h6 being the smalles

HTML elements - Defined by a start tag and we write some content in between and close it with end tag.

some of the elements will have no content and no end tags.
<hr> - horizontal line with no content and no end tag
<br> - empty element , line break.

Body - Only content inside the body section will be displayed.

Features of HTML
-----------------------
Simple and easy to use
Platform independant ( windows, mac, linux)
Case insentive language - but write in lower case for better readability and consistency.

Building blocks of HTML
-------------------------
Tags - Surrounds the content and apply meaning to it, we write using < and > brackets
attribute -- Provides extra information about the element
<tag attribute-name="attribute-value">
